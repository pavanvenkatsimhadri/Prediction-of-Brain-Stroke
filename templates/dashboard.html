About the Project

Brain Stroke Prediction is a Machine Learning based web application developed using Python Flask. The system predicts whether a patient has a low, medium or high risk of stroke based on their health information. No user data is stored in a database. The prediction is performed instantly and can be downloaded as a report.

Developed for MCA PG Final Year Project
